今天发看到一个条件判断

~~~ csharp
if( value = ( config ?? "" ) )
    config ?? ""
~~~

等同于

~~~ csharp
config=null?"":config;
~~~

意思是如果config为null那么就赋值””否则保持原值不变

举个例子：

~~~ csharp
int? i;
int x=i??5;// 如果i值为null那么给x赋值5，否则x值等于i的值
~~~

这个运算符只支持引用类型和Nullable类型。
